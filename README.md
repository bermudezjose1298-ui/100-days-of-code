## 🚀 Featured Project — Blackjack (Day 11)

The Day 11 capstone project is a fully functional Blackjack game built in Python.

**Features:**
- Full deck simulation with card drawing logic
- Dealer AI that draws until reaching 17+
- Win / lose / draw / blackjack detection
- Play again loop
- Clean terminal UI with ASCII art

**Concepts applied:** Lists, functions, conditionals, loops, randomisation, game state management.

## 🛠️ How to Run Any Project

1. Clone the repo:
   ```bash
   git clone https://github.com/bermudezjose1298-ui/100-days-of-code.git
   ```
2. Navigate to a day's folder:
   ```bash
   cd "Day 11"
   ```
3. Run the project:
   ```bash
   python main.py
   ```

> **Requirements:** Python 3.x — no external libraries needed for Days 1–14.
